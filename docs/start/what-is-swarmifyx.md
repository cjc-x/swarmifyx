---
title: 什么是 Swarmifyx？
summary: 面向自治 AI 公司的控制平面
---

Swarmifyx 是面向自治 AI 公司的控制平面。它是一套基础设施骨架，让 AI 劳动力可以在具备结构、治理和责任边界的前提下运行。

一个 Swarmifyx 实例可以运行多家公司。每家公司都拥有员工（AI 代理）、组织结构、目标、预算和任务管理，也就是一家真实公司所需的核心要素，只不过这里的“操作系统”变成了真正的软件。

## 要解决的问题

普通任务管理软件远远不够。当你的整个劳动力都是 AI 代理时，你需要的不只是一个待办列表，而是一整家公司的 **控制平面**。

## Swarmifyx 做什么

Swarmifyx 是一家 AI 代理公司的指挥、沟通与控制中心。你可以在这里：

- **像管理员工一样管理代理**：招聘、组织，并追踪每个代理在做什么
- **定义组织结构**：让代理在明确的组织架构中协作
- **实时追踪工作**：随时查看每个代理当前在处理什么任务
- **控制成本**：为每个代理设置 token 预算，跟踪消耗和 burn rate
- **围绕目标对齐工作**：让代理知道自己的工作如何服务更大的使命
- **治理自治能力**：通过董事会审批、活动审计和预算限制维持可控性

## 两个层次

### 1. 控制平面（Swarmifyx）

这是中央神经系统。它负责管理代理注册表与组织架构、任务分配与状态、预算与 token 消耗、目标层级以及心跳监控。

### 2. 执行服务（适配器）

代理在外部运行，并把结果回传到控制平面。适配器负责连接不同执行环境，例如 Claude Code、OpenAI Codex、shell 进程、HTTP webhook，或任何可以调用 API 的运行时。

控制平面本身并不执行代理，它只负责编排。代理可以运行在任何地方，再把执行情况汇报回来。

## 核心原则

你应该能够一眼看懂整个公司当前的状态：谁在做什么、花了多少钱、系统是否运转正常。
